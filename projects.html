Lab Rats is a 2D shooter and rhythm game hybrid I helped developed in EECS 494 - Game Design. As a team of four, we designed the concept from scratch and underwent weeks of playtesting and rounds of bugfixing to polish the game. We kept detailed release notes on an indie game developing forums and showcased our game at the end of the semester.

Insta485 is an Instagram clone I worked on in EECS 485 as a way to practice implementing client-side and server-side dynamic pages. We created multiple versions of the website with similar features but different implementations using React, Flask, and REST APIs

Fakebook is a simplified version of Facebook that was used in EECS 484 to learn SQL and database management. Throughout the semester, I created visual representations of schema used to create the database structure, then constructed queries that performed various functions crucial to a working social media database implementation

Over the summer of 2022, I interned at Tyler Technologies in Troy, Michigan where I contributed to an AGILE development team working on a cashiering application for the public sector. In this internship, I got experience with AGILE methodology, Angular HTML and C# backend. Interns also performed support work in bugfixing existing issues with the application.

As an introduction to Angular framework, I created a card tracking website for the trading card game Magic: The Gathering. This site used existing API and allowed users to search cards by criteria and customize lists of cards as decks. The project used Angular 3 and used a C# backend server to facilitate saving preferences.
